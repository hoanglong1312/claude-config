# CLAUDE.md — Personal Project

## Nguyên Tắc
- Ưu tiên đơn giản, bền vững
- Không over-engineer giải pháp
- Focus vào thói quen, không chỉ kết quả

## Quy Trình
- Bắt đầu từ mục tiêu rõ ràng
- Chia nhỏ thành hành động hàng tuần
- Review tiến độ định kỳ

## Memory Structure

```
goals.md                   ← mục tiêu, metric thành công, deadline
weekly-review.md           ← review tiến độ theo tuần
notes.md                   ← ghi chú nhanh
context/decisions.md       ← quyết định/cam kết cá nhân đã chốt
```

Ưu tiên ghi quyết định và next action ngắn, tránh biến project cá nhân thành hệ thống quá nặng.
